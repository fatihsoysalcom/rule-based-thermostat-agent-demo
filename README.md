# Rule Based Thermostat Agent Demo

This example demonstrates a simple rule-based system for maintaining a desired temperature range, simulating a basic thermostat. It illustrates a scenario where a complex AI agent might be an overkill, highlighting that traditional, straightforward algorithms can often be more efficient and appropriate for well-defined problems with predictable environments, aligning with the article's theme of understanding AI agent limitations.

## Language

`python`

## How to Run

Save the code as `main.py` and run it from your terminal using `python main.py`.
